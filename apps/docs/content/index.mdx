# Tesserix Design System

Cross-platform component system for web and React Native, organized as workspace packages.

## What is documented here

- `@tesserix/web`: web components, themes, Tailwind preset
- `@tesserix/native`: React Native component library (Expo-first)
- `@tesserix/tokens`: platform-agnostic design tokens
- `@tesserix/icons`: web/native icon exports + custom icons
- `@tesserix/hooks`: reusable React hooks
- `@tesserix/utils`: cross-platform utilities and providers

## Monorepo structure

```txt
apps/
  docs/

packages/

  native/
```
