# @tesserix/utils

Cross-platform utility package for app-level concerns.

## Install

```bash
pnpm add @tesserix/utils
```

## Exports

- `ErrorBoundary`
- `ThemeProvider`, `useTheme`
- `LoadingProvider`, `useLoading`
- `LoadingSpinner`, `LoadingDots`, `LoadingOverlay`, `LoadingSkeleton`

## Example

```tsx
import { ErrorBoundary, ThemeProvider, LoadingProvider } from '@tesserix/utils'
```
