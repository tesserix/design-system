# @tesserix/native

Expo-first React Native component package.

## Install

```bash
pnpm add @tesserix/native @tesserix/tokens @tesserix/hooks @tesserix/utils
```

Required peers:

```bash
pnpm add react react-native
```

## Highlights

- Layout, form, feedback, and data-display components
- Uses shared tokens from `@tesserix/tokens`
- Supports Expo and bare React Native projects
