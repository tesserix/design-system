# @tesserix/web

Web component package for React 19 with Tailwind 4 and Radix primitives.

## Install

```bash
pnpm add @tesserix/web @tesserix/tokens
```

Required peers:

```bash
pnpm add react react-dom tailwindcss
```

## Key exports

- Main entry: `@tesserix/web`
- Styles: `@tesserix/web/styles`
- Themes: `@tesserix/web/themes/default` (and other theme files)
- Tailwind preset: `@tesserix/web/tailwind`

## Setup summary

1. Add the Tailwind preset from `@tesserix/web/tailwind`.
2. Include package `dist` in Tailwind content globs.
3. Import `@tesserix/web/styles` once at your app root.
